# Granite Speech Pipeline

Automatic speech recognition (ASR) and automatic speech translation (AST) using IBM Granite Speech models.

## Features

- Transcribe speech to text
- Translate speech to French, German, Spanish, or Portuguese
- Supports multiple audio formats (WAV, MP3, FLAC, OGG, M4A, WMA)
- Download transcriptions as TXT or SRT

## Installation

```bash
python3.12 -m venv streamlit_env
source streamlit_env/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
streamlit run streamlit_app.py
```
